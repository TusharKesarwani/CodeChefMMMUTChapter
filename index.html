What do we do

Coding Contests

We organize coding contests every year for new upcoming batch students. Through these contests students check their coding skills and problem solving skills. We also provide goodies to the winners of the contests.

Problem Solving Sessions

Problem Solving Sessions is for those students who wants to upskill thier problem solving skills. In this session we all connecting with each other at one platform and targeting some good DSA questions.

Training Sessions

Training Sessions is for those students who are new in the coding field and don't even know what is meant by programming languages.

Learning Sprints

Learning Sprints is for those students who want to make command in the programming languages like Java, C, C++ and Python.
